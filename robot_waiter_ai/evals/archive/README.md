# Archived Mock Evaluation Files

This directory contains historical mock/simulated evaluation files from development. They are preserved for reference only and should **not** be confused with real model audit outputs generated during Sprint 1.

## Contents

*   `mock_eval_50_questions.py`: Script to generate mock responses or run simple evaluations against simulated prompts.
*   `mock_qwen_50_soru_test_sonuc.md`: Simulated/mock 50-question test results showing a past simulated run.

For actual Sprint 1 Quality Audit reports, please refer to the corresponding active runner and output directories in `robot_waiter_ai/evals/` or `reports/`.
